DISTINCT: 고유한 값만 보고 싶을 때 사용. UNIQUE한 값만 알고 싶은 경우 사용  
ex) COUNT(DISTINCT user_id) AS dau -> 이런 데이터를 저장하는 곳에 접속 기록, 이벤트 로그가 여러 ROW 존재 => UNIQUE => DISTINCT
