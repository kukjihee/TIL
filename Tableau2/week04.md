# 학습 내용 정리

## 30. 계층

데이터의 구조적 관계를 단계적으로 표현하는 기능



## 31. 집합   

특정 조건이나 선택을 만족하는 데이터의 하위 집합



## 32. 결합집합

두 개 이상의 집합을 비교하여 교집합, 합집합, 차집합 등을 표현하는 기능   
=> 특정 조건을 동시에 만족하는 대상을 빠르게 식별할 수 있음



## 33. 계산된 필드

데이터 원본에 없는 값을 사용자 정의 계산식으로 생성하는 필드   
=> ex) 수익률 = SUM([수익]) / SUM([매출])


## 34. 행수준계산

Row-Level Calculation: 데이터베이스의 각 행에 대해 계산이 이루어지는 방식    
=> 집계 전에 미리 계산되는 특징



## 35. 집계계산

Aggregate Calculation: 여러 행의 데이터를 집계(합계, 평균, 최대, 최소 등)한 후 계산



## 36. 테이블계산

뷰(View)에 표시된 결과값을 바탕으로 추가로 계산하는 기능   
=> 데이터 소스가 아니라 뷰의 테이블 방향에 따라 결과가 달라짐



## 37. 퀵테이블계산(1)

자주 쓰이는 테이블 계산을 바로 적용할 수 있는 기능   

ex) 누계, 차이, 비율 차이, 구성 비율, 순위, 백분위수



## 38. 퀵테이블계산(2)

* 이동 평균(Moving Average): 일정 구간의 평균값으로 추세 확인

* YTD 총계: 연초부터 현재까지의 누적 합계

* 전년 대비 성장률: 동일 기간의 전년 데이터와 비교

* YTD 성장률: 연초~현재 매출이 전년 동기 대비 얼마나 성장했는지 측정
