# 학습 내용 정리

## 48. 워크시트 서식(2)

* 폰트, 색상, 눈금선, 테두리, 툴팁, 레이블, 축
* 전체 대시보드에 일관된 스타일 유지 중요



## 49강. 대시보드패널

* 여러 워크시트를 한 화면에 결합
* 화면 왼쪽 패널에서 객체를 드래그해 배치



## 50. 대시보드 구성방식

<!-- 알게 된 점을 적고, 아래 질문에 답해보세요 :) -->

> **🧞‍♀️ 부동과 바둑판식 방식을 차이를 중점으로 기술해보세요**
* 부동
: 자유로운 위치 배치, 겹치기 가능

* 바둑판식
: 격자형 자동 정렬, 세밀한 배치는 어려움   




## 51. 대시보드 컨테이너
* 가로 컨테이너: 좌우 배치   
* 세로 컨테이너: 상하 배치   
* 컨테이너 단위로 크기 조정 및 그룹 이동 가능   
* 균등 비율 유지 및 반응형 구성에 필수


## 52. 레이아웃 패널
* 선택한 객체의 크기·위치·여백·테두리 조정   
* 그래프 간 간격 조절로 깔끔한 대시보드 구성

## 53. 필터 동작

* 대시보드 → 동작 추가 → 필터
* ex) 지역 선택 → 해당 지역 데이터만 다른 차트에 표시   



## 54. 대시보드 하이라이터 동작
* 데이터의 특정 항목을 강조 표시하는 기능
* 대시보드 → 동작 추가 → 하이라이트


## 55. 대시보드 URL

* 동작 추가 → URL → 대상 링크 지정   



## 56. 대시보드 시트에 이동 동작
* 클릭 시 다른 시트로 이동(Drill Down)
* 동작 추가 → 시트로 이동
* 요약-상세 분석 흐름 구성에 적합

## 57. 매개변수 변경 동작
* 동작 추가 → 매개변수 변경
