# 학습 내용 정리

## 58. 집합값 변경

기존의 집합 멤버를 수정하거나 새로 추가/제거하는 기능으로, 데이터 그룹화 기준을 동적으로 조정 가능   



## 59강. 스토리패널

여러 시트와 대시보드를 순차적으로 연결해 하나의 스토리라인(프레젠테이션) 을 구성하는 기능



## 60. 스토리

데이터 시각화를 활용해 “분석 과정 → 인사이트 → 결론”의 흐름을 전달하는 기능



## 61. 대시보드 탐색

탐색 기능을 통해 사용자가 직접 클릭하거나 필터를 조작하며 원하는 데이터를 탐색 가능



## 62. 태블로 단추

대시보드 내에서 다른 시트, 대시보드, 스토리로 이동할 수 있는 네비게이션 도구   



## 63. 막대그래프 드릴다운

드릴다운은 시각화에서 상위 수준의 데이터를 클릭해 세부 수준으로 내려가는 분석 방식



## 64. 트리맵 드릴다운

계층적 데이터를 사각형으로 표현하는 시각화



## 65. 파이 차트 드릴다운

파이 차트는 항목이 많을 경우 가독성이 떨어질 수 있어 드릴다운 구조로 단계적 분석이 효과적



## 66. 지도 드릴다운

지도 시각화에서 지역 단위를 클릭해 상위 행정구역 → 하위 구역으로 세분화하여 볼 수 있다.

# 2️⃣ 실습형 과제 (마지막 과제)

>  **🧚 지금까지 배운 Tableau의 기능 (집합값 변경, 드릴다운, 대시보드, 스토리 등)을 활용하여, 자신이 선택한 데이터로 하나의 대시보드를 만들어 스토리 시각화를 제작해보세요.**

~~~
과제 가이드라인

1. 데이터 선택
- 자유 주제 : Kaggle, 공공 데이터 포털 가능
- 단, 데이터셋에 최소 3개 이상의 수치형 변수, 2개 이상의 범주형 변수 포함

2. 시각화 구성 (다음 중. 가지 이상 기능 사용하기)
- 막대그래프/라인차트/트리맵/파이차트 중 1개 이상
- 드릴다운 기능 1개 이상 (어떻게 적용했는지도 설명하셔야 합니다.)
- 대시보드 또는 스토리 형태로 구성
- 단추 (Button) 또는 필터 기능을 활용하기 

3. 분석 스토리라인 작성해보기
- 이 시각화는 어떤 문제의식 또는 인사이트를 전달하려고 하는가?
- 데이터가 보여주는 사실과 해석은 무엇인가?
- 태블로의 어떤 기능들을 배웠었고, 이를 어떻게 적용했는가?
- 추가적으로 필요한 데이터나 맥락은 무엇일까?
~~~

~~~
여기에 분석 스토리라인을 작성하세요. 
~~~
